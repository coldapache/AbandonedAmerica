How to Contribute

Help build the national database of abandoned and blighted properties

For Humans

You know your city. You drive past that boarded-up building every day. Here's how to add it:

  • Open Google Maps and find the property. Right-click to get the coordinates. Copy the URL (Street View preferred).
  • Look up the property records on your county assessor's website. Search "[your county] property tax records" to find the owner and assessed value.
  • Check if it's already in the database by searching the sidebar on this page.
  • Fork the repo on GitHub, add a row to the CSV, and open a pull request.

For AI Agents (Claude Code, Codex, etc.)

Clone the repo and use the built-in slash commands:

  • /hunt-properties Detroit, MI - Search for and add abandoned properties in an area
  • /verify-property all - Visually verify existing properties via Google Maps
  • /repair-data - Fill in missing owner, assessment, and source data
  • /validate-csv - Run a full data quality check

What Counts as ABNC?

Abandoned, blighted, and non-contributing properties include:

  • Buildings that have been empty for a long time with no signs of returning to use
  • Properties that are visibly deteriorating (boarded windows, collapsed roof, overgrown)
  • Officially condemned by local government
  • Buildings where the business closed and the building sits empty with no new tenant
  • Structures that have been demolished, leaving empty lots

If you're unsure, look for the signs: no lights, no cars, no maintenance, weeds growing through the parking lot, weathered "For Sale" signs, faded signage. You know it when you see it.

Data Sources

  • County assessor/tax records - owner names, assessed values
  • City code enforcement - condemned and blighted property lists
  • Google Maps Street View - visual confirmation
  • Local news - articles about blight, demolitions, abandoned buildings
  • Your own eyes - if you live there, you know where the abandoned buildings are
